# MRS Agro Chemicals

A modern, responsive corporate website developed for **MRS Agro Chemicals**, a pesticide and agricultural chemicals company.

## Features

- Modern & Responsive UI
- Product Showcase
- Company Profile
- About Us
- Contact Form
- Franchise Inquiry
- Admin Dashboard
- Multi-language Support (English & Urdu)
- Mobile Friendly
- Fast Performance

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- SQLite
- Vercel

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Deployment

Optimized for deployment on Vercel.
